Getting the code

Play framework is hosted on Github using the Git version control system. You can get the code with the following command:

git clone git://github.com/playframework/Play20.git

If you're looking for the code for the previous Play 1.x series, it's available in another repository:

git clone git://github.com/playframework/play.git

Licence

The Play framework source code is released under the Apache 2 licence.

Reporting bugs

We use Lighthouse as our issue tracker. Bug reports are incredibly helpful, so please feel free ot take the time to report bugs and request features. And we’re always grateful for patches to Play’s code. Indeed, bug reports with attached patches will get fixed far quicker than those without.

Contributing

The best ways to contribute are by finding and reporting bugs, writing tests for bugs, and improving the documentation. As we mentioned: we’re always grateful for patches to Play!

If you want to change the code, be careful to follow our design decisions. Be especially careful not to break compatibility with modules and older versions, or to increase complexity without a really solid reason.

Have a patch? Check the contributor guide.
